# TurfScore

This is the monorepo starter for TurfScore. It contains a React frontend (Vite) and an Express backend (Node.js).

## Installation

1. Ensure you have Node.js (v18+) and npm installed.
2. Clone this repository.
3. Install dependencies from the root directory:

```bash
npm install
```

## Running Locally

You can start both the frontend and backend servers simultaneously from the root directory:

```bash
npm run dev
```

- **Frontend:** http://localhost:3000
- **Backend:** http://localhost:5001
